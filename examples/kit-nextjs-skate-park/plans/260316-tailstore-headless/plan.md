# Plan: Tailstore Headless E-commerce
Created: 2026-03-16
Status: 🟡 In Progress

## Overview
Chuyển đổi Tailstore HTML sang Next.js App sử dụng Sitecore JSS và OrderCloud. Áp dụng Atomic Design để tối ưu khả năng biên tập cho Content Author.

## Tech Stack
- **Frontend:** Next.js 15, Tailwind CSS.
- **CMS:** Sitecore XM Cloud (Content SDK).
- **Commerce:** OrderCloud SDK.
- **Styling:** Tailwind CSS (Atomic Utility).
